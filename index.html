Overview

What is PDX?

PDX is a text-first document format and editor that puts real, editable text at the core. No more fighting with canvas-based layouts or struggling to edit "PDF documents". PDX is designed to replace the pain of PDF editing with a clean, simple, and powerful alternative.

Benefits

Why PDX?

PDX brings together the best of document editing with the simplicity you deserve. Here's why you should choose PDX over traditional PDF editors.

  • Real text, not canvas - Edit documents like you edit code or plain text
  • Built-in editor application - No need for separate tools or complex software
  • Open-source - Community-driven, transparent, and free forever
  • Fast and lightweight - No bloat, no unnecessary features, just what you need
  • Easy to edit and share - Simple file format that works everywhere
Application

Editor Application

PDX is more than just a file format - it's a complete application. The built-in editor provides a seamless workflow for creating, editing, and managing your documents. No plugins, no extensions, no hassle.
